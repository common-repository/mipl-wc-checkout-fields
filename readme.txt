Customize WooCommerce Checkout Fields, Create Group of custom fields & Update default checkout fields.


== Description ==

Customize WooCommerce checkout fields by creating groups of custom fields for specific products or categories, with the ability to repeat fields based on product quantity. Additionally, update and tailor default checkout fields to suit your needs.


= Features =

- **Custom Field Groups:** Create and manage custom checkout field groups.
- **Target Specific Products/Categories:** Assign fields to specific products or categories.
- **Quantity-Based Repetition:** Repeat field groups based on product quantity.
- **Order & Email Integration:** Display fields on the order edit page and in email templates.
- **Diverse Input Types:** Support for various input types (text, select, textarea, radio, checkbox, email, phone, time, date, color, datetime).
- **Customizable Default Fields:** Update and customize default checkout fields.



== Installation ==

1. Upload plugins folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create Checkout Fields Group and Create Required Fields as per the Product.
4. Make Field Group Settings.
